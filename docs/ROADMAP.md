# Roadmap

## 0.1 — deterministic foundation

- Codex, Claude Code, Cursor, and GitHub Copilot adapters
- File-level context trace
- Comparison UI
- Token estimates and deterministic findings
- JSON schema-compatible report

## 0.2 — verification harness

- Pinned client integration tests where licensing and automation permit
- Adapter changelog and compatibility matrix
- Snapshot tests for official documentation examples

## 0.3 — developer workflow

- GitHub Action with PR comments
- SARIF export
- VS Code extension
- Report diff between branches

## Non-goals

- Executing instructions or shell commands
- Claiming to reproduce hidden system prompts
- Uploading private repositories to a hosted service
- Using an LLM as the default conflict detector
